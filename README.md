Final Project for CSCA 5622 - Introduction to Machine Learning: Supervised Learning

```bash
python3 -m venv venv
source venv/bin/activate
python3 -m pip install --upgrade pip
python3 -m pip install -r requirements.txt
```

```bash
python3 main.py
```
